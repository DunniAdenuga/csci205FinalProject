# Sudoku Solver
Create Sudoku puzzles automatically, or enter them into the program manually.  Solve Sudoku puzzles with intuitive cues that point out errors.  For harder puzzles, choose from a variety of algorithms that will help you solve the puzzle.

## Building
This directory contains a build.xml file.  If ant is installed, type

    ant jar

to build a JAR file in the dist directory.  Alternatively, this project can be opened and built in netbeans.
